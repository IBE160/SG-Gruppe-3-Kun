# Welcome to HMSREG Documentation

Welcome to the official documentation for the **HMSREG Documentation Chatbot**. This project aims to simplify access to technical documentation using AI-powered assistance.

## Getting Started

To get started with the HMSREG chatbot, navigate through the links on the left sidebar. You can explore:

*   **Introduction**: This page.
*   **Architecture**: High-level system design.
*   **Authentication**: How we handle user security.

## Features

*   **Role-Based Access**: Tailored responses for Developers, PMs, and Architects.
*   **Interactive Chat**: Ask questions in natural language.
*   **Multi-language Support**: English and Norwegian.
